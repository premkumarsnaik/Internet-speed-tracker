# Internet Speed Tracker

A simple Python-based tool to test and log local network performance (Download, Upload, and Latency) directly to a CSV file.

## Features
* Measures true local Wi-Fi speeds instead of cloud data center speeds.
* Logs timestamps, download speed (Mbps), upload speed (Mbps), and ping (ms).
* Automatically stops execution after a designated duration.
